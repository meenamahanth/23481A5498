# Campus Notification Management System

This project is a full-stack notification management system built using React, Node.js, and Express. The application fetches notifications from an external API and allows users to view, filter, paginate, and prioritize notifications in a clean and responsive interface.

The project is divided into three modules:

- logging_middleware – reusable middleware for logging requests
- notification_app_be – backend API built with Express
- notification_app_fe – frontend application built with React and Material UI
